
# VANNA NSK — Премиум сайт ремонта ванных комнат

## Дизайн-система: Liquid Glass + Dark Premium

Тёмный фон (#050505) с радиальными зелёными градиентами, grain-текстурой и эффектом глубины. Все интерактивные элементы — полупрозрачные стеклянные карточки с backdrop-blur, неоновым зелёным свечением (#00FF88) и плавными анимациями. Шрифт Inter для чёткой премиальной типографики.

---

## Страница 1: Главная

### Фиксированный Glass Header
- Полупрозрачная навигация с размытием фона
- Логотип «VANNA NSK» с неоновым акцентом
- Меню: Услуги → Наши работы → Цены → Контакты
- Кнопка «Заказать замер» с glow-эффектом

### Hero-секция (вау-эффект)
- Полноэкранный блок с фоновым изображением ванной (затемнение)
- Floating стеклянный контейнер с заголовком «Ремонт ванных комнат нового уровня»
- Две кнопки: неоновая primary + стеклянная outline
- Три glass-иконки: «Срок от 7 дней», «Фиксированная смета», «Гарантия до 3 лет»

### Блок услуг — Glass Grid (6 карточек)
- Ремонт под ключ, Санузел под ключ, Замена плитки, Реставрация ванны, Душевые зоны, Премиум дизайн
- Hover: усиление blur, glow по краям, лёгкое увеличение

### До / После — интерактивный слайдер
- Стеклянная рамка с drag-слайдером сравнения
- Placeholder-изображения (вы загрузите свои фото позже)

### Процесс работы (5 шагов)
- Горизонтальная линия со стеклянными шагами: Замер → Смета → Договор → Ремонт → Сдача

### Премиум CTA
- Большой glass-контейнер: «Готовы к ванной вашей мечты?» + кнопка «Получить бесплатный расчёт»
